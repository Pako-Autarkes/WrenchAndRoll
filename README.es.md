# WrenchAndRoll
Guía para novatos. Pon a punto FreePascal (Geany/Lazarus) y C/C++ en Linux, Windows y macOS.
Sin carnet ni gaitas — coge la llave inglesa y a rodar.
 
![WrenchAndRoll](WrenchAndRoll.jpg)
 
**WrenchAndRoll** — el kit del mecánico callejero para compilar algunos de mis proyectos.  
Sin rollos. Un solo setup. Sustituto directo del "Colega, que no sé cómo compilar esto". Sin el lío de dependencias, sin dramas, sin el RTFM.
 
**Doble cadena de herramientas:** Free Pascal y C/C++. Misma especificación. Dos lenguajes. Builds de verdad.

## ¿Qué es esto?
 
¡Venga, novato! ¿No sabes distinguir un compilador de un carburador?  
Este repo le mete caña a tu máquina **Linux**, **Windows** o **macOS** para que puedas compilar algunos de mis proyectos de FreePascal y C/C++. Sin experiencia previa — coge la llave y a rodar.
 
---
 
## Cadena de herramientas por plataforma
 
| SO | Free Pascal | Free Pascal (GUI) | C/C++ | IDE / Editor |
|:---|:---|:---|:---|:---|
| **Linux** | `fpc` + `Geany` | `fpc` + `Lazarus` | `gcc` / `g++` + `Geany` | Lazarus + Geany |
| **Windows** | `fpc` + `Geany` | `fpc` + `Lazarus` | `gcc` / `g++` + `Geany` | Lazarus + Geany |
| **macOS** | `fpc` + `Geany` | `fpc` + `Lazarus` | `clang` / `clang++` + `Geany` | Lazarus + Geany |
 
> *(Si Geany no es lo tuyo — Eres más de Notepad++, Kate, o lo que uses — ya vas sobrao, colega. Sigue a lo tuyo, esta guía no es para ti.)*
